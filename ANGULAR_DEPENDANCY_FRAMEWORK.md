# Angular full-stack dependancy install using npm

1. package.json file -
`` npm init ``

2. Express -
`` npm install --save express ``

3. Body-Parser
`` npm install --save body-parser ``

4. PG
`` npm install --save pg ``

5. Angular One
`` npm install --save angular ``

6. Angular-Route One
`` npm install --save angular-route
 ``

7. Install mongoose
(if using MongoDB)
`` npm install --save mongoose ``
